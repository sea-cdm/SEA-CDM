Experiment

A table that represents the experiment as part of the SEA CDM. This table serves as a central node and contains information.

Experiment Properties

Field Description
idExperiment An internal ID that corresponds to an instance of an experiment.
StudyID An ID that corresponds to a study an experiment belongs to.
ExperimentControl A binary flag used to indicate if an experiment row is used as part of an experimental control or not.
ExperimentType A string used to describe the type of an experiment used. E.g. “vaccine challenge” and “immune response”.
ExperimentTypeID A string used to match Experiment Type to an ontology ID.
ForeignID A foreign ID used to represent the source of an experiment from another source. Must be paired with Foreign Source.
ForeignSource A string representing the Foreign Source for an experiment.
Comments A string used as a placeholder for extra words.
