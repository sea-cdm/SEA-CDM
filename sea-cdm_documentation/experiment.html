Experiment

A table that represents the experiment as part of the SEA CDM. This table serves as a central node for the SEA model. An Experiment can include clinical visits or experimental controls.

Experiment Properties

Property Description Usage
experiment_id An primary key that corresponds to an instance of an experiment. "EXPER_1"
study_id A foreign key that corresponds to a study an experiment belongs to. "STUDY_1"
documentation_id A foreign key that corresponds to a document that describes the protocol used as part of this experiment. "DOCU_1"
experiment_control A binary flag used to indicate if an experiment row is used as part of an experimental control or not. "True", "False"
experiment_type A string used to describe the type of experiment performed. "Clinical Visit", "Experimental"
experiment_type_id A string used to match experiment_type to an ontology ID. "OBI_0500000"
reference_id A foreign ID used to represent the source of an experiment from another source. Must be paired with reference_source. "ARM_0000002"
reference_source A string representing the reference source for an experiment. ImmPort
comments A string used as a placeholder for additional notes or remarks.
