Experiment

A table that represents the experiment as part of the SEA CDM. This table serves as a central node and contains information. Experiments can include clinical visits within the SEA CDM.

Experiment Properties

Property Description
experiment_id An internal ID that corresponds to an instance of an experiment.
study_id An ID that corresponds to a study an experiment belongs to.
experiment_control A binary flag used to indicate if an experiment row is used as part of an experimental control or not.
experiment_type A string used to describe the type of an experiment used. E.g. “vaccine challenge” and “immune response”.
experiment_type_id A string used to match Experiment Type to an ontology ID.
reference_id A foreign ID used to represent the source of an experiment from another source. Must be paired with reference_source.
reference_source A string representing the reference source for an experiment.
comments A string used as a placeholder for extra words.
