Study

A study is a planned process that is intended to answer a scientific question through multiple types of experiments or observational studies.

Study Properties

Property Description Usage
study_id A primary key used to represent a study. "STUDY_1"
study_name A string representing the name of a study. "OSEAN-DB Vaccine Gene Response Study"
study_description A string representing a short description of a study. This is set to 500 characters or less and is intended to illustrate the key question of a study. "A study to analyze difference in vaccine gene response data from influenza using data in OSEAN"
study_type A string representing the type of a study. "Clinical Investigation", "Experimental Study", "Meta-Analysis"
study_type_id An ontology string representing the type of a study. OBI_0500000
study_focus A string representing the intended domain a study is about. "Immune Response", "Infection Response", "Vaccine Response"
study_focus_id An ontology string representing the study focus. BFO_0000015
study_subject A string representing the intended subject matter shown by a study; this can be a process or material entity. "Influenza", "tuberculosis"
study_subject_id A An ontology string representing the intended subject matter shown by a study. BFO_0000002
reference_id A foreign ID used to represent the source of a study from another source. Must be paired with reference_source. "SDY562"
reference_source A string representing the reference source for a study. ImmPort
Comments A string used as a placeholder for extra words or comments.
