Study

A study is a planned process that is intended to answer a scientific question through multiple types of experiments or observational studies.

Study Properties

Property Description
study_id A primary key used to represent a study.
study_name A string representing the name of a study.
study_description A string representing a short description of a study. This is set to 500 characters or less and is intended to illustrate the key question of a study.
study_type A string representing the type of a study. This is typically 'Experiment', 'Observation', or 'Meta Analysis'.
study_type_id An ontology string representing the type of a study.
reference_id A foreign ID used to represent the source of a study from another source. Must be paired with ForeignSource.
reference_source A string representing the reference source for a study.
Comments A string used as a placeholder for extra words or comments.
