Study

Study Properties

Property Description
idStudy An internal ID used to represent a study.
StudyName A string representing the name of a study.
StudyType A string representing the type of a study. This is typically 'Experiment', 'Observation', or 'Metanalysis'.
StudyTypeID An ontology string representing the type of a study.
ForeignID A foreign ID used to represent the source of a study from another source. Must be paired with ForeignSource.
ForeignSource A string representing the Foreign Source for a study.
Comments A string used as a placeholder for extra words or comments.
