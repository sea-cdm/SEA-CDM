Materials

A material is a catch-all to represent any material entity that has a dose used as part of a material experiment or assay that does not belong within the organism table.

Materials Properties

Property Description
material_idA primary key used to identify a specific material.
material_nameA string used to describe the material name.
material_name_idA string used to identify the ontology id that is associated with the material name.
organizationA string used to identify the organization responsible for the creation or refinement of a material.
ForeignIDA string used to identify an ID that is used for a material within a different database or source.
reference_idA string used to identify an external reference for the group.
reference_sourceA string used to identify the source of the ForeignID.
