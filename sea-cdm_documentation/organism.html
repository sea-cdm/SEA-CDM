Organism

This table contains a list of all organisms listed in an experiment as part of the SEA CDM. Each organism is listed as an instance within a dataset; the same organism from different studies will not be linked unless there is a connection between foreign_id values. An organism is a necessary component for a biological experiment.

Organism Properties

Property Description
organism_id An internal ID that corresponds to an instance of an organism.
group_id An internal ID that corresponds to which group an organism belongs to; organisms that do not belong to a group are assigned a group value of 1 (to represent a singleton).
organism_species A string representing the species an organism belongs to; this is a common name that is listed.
organism_species_id A string representing the species an organism belongs to; this species is identified as equivalent to “OBI_0100026”.
organism_type A string representing the subtype an organism belongs to; this subtype could correspond to a human race or ethnicity, animal breed, or microbial strain.
organism_mutation A string representing an organism that belongs to have an atypical (to wild type of OrganismRace) mutation.
organism_type_id A string representing the ontology ID that corresponds to an OrganismType.
organism_age A double that represents the organism age; this is the lowest age listed for the instance of an organism. This must be paired with an OrganismAgeUnit and OrganismAgeID.
organism_age_unit A string representing the units for an organism's age.
organism_age_unit_id A string representing the ontological ID used for organism_age_unit.
organism_sex A string representing an organism’s biological sex. Currently, four values are 0/1/2/3 representing male/female/both/neither.
organism_sex_id A string representing the ontological ID used for organism_sex.
