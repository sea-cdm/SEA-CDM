Organism

This table contains a list of all organisms listed in an experiment as part of the SEA CDM. Each organism is listed as an instance within a dataset; the same organism from different studies will not be linked unless there is a connection between foreignIDs.

Organism Properties

Property Description
idOrganism An internal ID that corresponds to an instance of an organism.
idGroup An ID that corresponds to which group an organism belongs to; organisms that do not belong to a group are assigned a group value of 1 (to represent a singleton).
OrganismSpecies A string representing the species an organism belongs to; this species is identified as equivalent to “OBI_0100026”.
OrganismType A string representing the subtype an organism belongs to; this subtype could correspond to a human race or ethnicity, animal breed, or microbial strain.
OrganismTypeID A string representing the ontology ID that corresponds to an OrganismType.
OrganismAge A double that represents the organism age; this is the lowest age listed for the instance of an organism. This must be paired with an OrganismAgeUnit and OrganismAgeID.
OrganismAgeUnit A string representing the units for an organism's age.
OrganismAgeID A string representing the ontological ID used for OrganismAgeUnit.
OrganismSex An integer representing an organism’s biological sex. Currently, four values are 0/1/2/3 representing male/female/both/neither.
