Organism

This table contains a list of all organisms listed in an experiment as part of the SEA CDM. Each organism is listed as an instance within a dataset; the same organism from different studies will not be linked unless there is a connection between foreign_id values. An organism is a necessary component for a biological experiment.

Organism Properties

Property Description Usage
organism_id An internal ID that corresponds to an instance of an organism. "ORGO_1"
experiment_id A foreign key that corresponds to which experiment an organism belongs to. "EXPER_1"
group_id An foreign key that corresponds to which group an organism belongs to; organisms that do not belong to a group are assigned a group value of 1 (to represent a singleton). "GROUP_1"
species A string representing the species an organism belongs to; this is a common name that is listed. "Human"
species_id A string representing the species an organism belongs to; this species is identified as equivalent to “OBI_0100026”. "NCBITaxon:9606"
organism_race A string representing the subtype an organism belongs to; this subtype could correspond to a human race or ethnicity, animal breed, or microbial strain. Leave blank for non-humans “White”, “European”, “HA1 mutants”
organism_race_id A string representing the ontology ID that corresponds to an organism_race.
organism_lineage A string representing the subtype an organism belongs to; this subtype could correspond to animal breed, or microbial strain. “BALB/C”, “Influenza A Virus Brisbane/A1202”
organism_lineage_id A string representing the ontology ID that corresponds to an organism_lineage.
organism_age A double that represents the organism age; this is the lowest age listed for the instance of an organism. This must be paired with an OrganismAgeUnit and OrganismAgeID. 46, 57
organism_age_unit A string representing the units for an organism's age. "Month", "Year"
organism_age_unit_id A string representing the ontological ID used for organism_age_unit. “UO_0000003”, “UO_0000004”,
organism_sex A string representing an organism's biological sex. "Male", "Female"
organism_sex_id A string representing the ontological ID used for organism_sex. “PATO_00000383”, “PATO_00000384”, “PATO_0000047”
reference_id A string representing the ID listed by the organism in the reference id.
reference_source A string representing the organism source for an ontology id “ImmPort”
comments A string representing comments about the organism class entry.
