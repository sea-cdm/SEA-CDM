Intervention

An intervention is a process where one organism is exposed to one material entity.

Intervention Properties

Property Description
idIntervention An internal ID used to identify an experiment intervention.
ExperimentID An ID that is used to link an experimental intervention to an Experiment.
OrganismID An ID that is used to link an experimental intervention to an Organism.
Material A string used to identify the material used as part of an intervention.
Dosage A string used to identify the dosage used of a material given to an organism.
DosageUnit A double used to identify the unit of a dosage.
DosageUnitID An integer used to identify the unitID mapped to the dosage.
Route A string used to identify the route of administration for an experiment intervention.
RouteID A string used to identify the ontology ID for route.
T0 Definition A string used to identify when T0 is given for intervention.
InterventionTime A double used to identify how long after the start of T0 an intervention occurred.
TimeUnit A string used to identify the unit for InterventionTime.
TimeUnitID A string used to identify the ontology ID for TimeUnit.
