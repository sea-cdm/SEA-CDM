Intervention

An intervention is a process where one organism is exposed to one material entity.

Intervention Properties

Property Description Usage
intervention_id A primary key used to identify an experiment intervention. "INTER_1"
experiment_id A foreign key that is used to link an experimental intervention to an Experiment. "EXPER_1"
organism_id A foreign key that is used to link an experimental intervention to an Organism. "ORGO_1"
material A string used to identify the material used as part of an intervention. "Influenza vaccine", "yellow fever virus", "Tamoxifen"
dosage A string used to identify the dosage used of a material given to an organism. 2
dosage_unit A double used to identify the unit of a dosage. "cc", "ml"
dosage_unit_id An integer used to identify the unit ID mapped to the dosage. IAO_0000003
intervention_route A string used to identify the route of administration for an experiment intervention. "Intranasal", "Intravenous"
intervention_route_id A string used to identify the ontology ID for route. VO_0000574, BFO_0000049
t0_definition A string used to identify when T0 is given for intervention. "Time of administration", "Time of study"
intervention_start A double used to identify how long after the start of T0 an intervention occurred. 7
time_unit A string used to identify the unit for intervention_start. "Day", "Month"
time_unit_id A string used to identify the ontology ID for time_unit. UO_000003
