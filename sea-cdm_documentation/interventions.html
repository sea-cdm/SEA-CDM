Intervention

An intervention is a process where one organism is exposed to one material entity.

Intervention Properties

Property Description
intervention_id An internal ID used to identify an experiment intervention.
experiment_id An ID that is used to link an experimental intervention to an Experiment.
organism_id An ID that is used to link an experimental intervention to an Organism.
material A string used to identify the material used as part of an intervention.
dosage A string used to identify the dosage used of a material given to an organism.
dosage_unit A double used to identify the unit of a dosage.
dosage_unit_id An integer used to identify the unit ID mapped to the dosage.
route A string used to identify the route of administration for an experiment intervention.
route_id A string used to identify the ontology ID for route.
t0_definition A string used to identify when T0 is given for intervention.
intervention_start A double used to identify how long after the start of T0 an intervention occurred.
time_unit A string used to identify the unit for intervention_start.
time_unit_id A string used to identify the ontology ID for time_unit.
