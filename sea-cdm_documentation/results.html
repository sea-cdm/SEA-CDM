Results

Results are data items generated from an assay, analysis of an assay, or an analysis on data that is derived from an assay.

Results Properties

Property Description Usage
results_id An internal ID used to identify assay results in a database. "RSULT_1"
experiment_id A foreign key used to identify the experiment an assay result is part of. "EXPER_1"
group_id An ID used to identify the group an assay result is about. Groups that are about individual organisms or samples are given a special ID. "GROUP_1"
sample_id A foreign key used to identify the sample a result is about. If used for a group, use “1” as placeholder value. "SAMP_1"
organism_id A foreign key used to identify the organism results is about. If used for a group, use “1” as placeholder value. "ORGO_1"
documenation_id A foreign key used to identify the document that contains the documentation for the sort. "DOCU_1"
analysis_type A string used to identify the current analysis used to generate a result.
original_assay_type A string used to identify the original assay type used to generate an assay. This is always an experimental assay.
original_assay_type_ID An ID used to identify the assay ID that an AssayType or OriginalAssayType is part of.
datatype A string used to identify the type of data the results is; either numeric, qualitative, etc.
datatype_ID An ontology ID that corresponds to the Datatype.
data_size A string used to identify the size of a document. Currently used as a string due to variable size from matrices or hierarchical data.
file_access A string used to identify the location or web link the results file is located. This is analogous to reference_source in other values.
file_type A string used to identify the file type extension the data is stored as (if accessible for computers). ".csv"
replications An integer that lists how many times an analysis is repeated to generate data; multiple measurements.
