Assay Results

A class representing the metadata that is generated from an experimental or statistical assay.

Assay Results Properties

Property Description
idResults An internal ID used to identify assay results in a database.
ExperimentID An ID used to identify the experiment an assay result is part of.
GroupID An ID used to identify the group an assay result is about. Groups that are about an individual organism are given a special ID.
SampleID An ID used to identify the sample an assay result is about.
DocumentID An ID used to identify the document that contains the documentation for the sort.
AssayType A string used to identify the current assay type used to generate an assay.
OriginalAssayType A string used to identify the original assay type used to generate an assay. This is always an experimental assay.
AssayTypeID An ID used to identify the assay ID that an AssayType or OriginalAssayType is part of.
Datatype A string used to identify the datatype information is stored as.
DatatypeID An ontology ID that corresponds to the Datatype.
Data_Size A string used to identify the size of a document. Currently used as a string due to variable size from matrices or hierarchical data.
Replications An integer that lists how many times an assay is repeated.
