Documentation

An unsorted table used to identify all documentation (data types, protocols, or results).

Documentation Properties

Property Description
documentation_id A primary key used to identify a specific documentation for a document as part of the study.
study_id A foreign key used to identify a study that Documentation belongs to.
document_name A string used to identify the name of a document.
reference_id A string used to identify a foreign ID used for a document. This is paired to reference_source.
reference_source A string used to identify the source of documentation that a reference_id belongs to.
documentation_type A string used to represent what the documentation is about; i.e. protocol, paper, results, etc.
person_id A string used to find a unique identifier of a person (if available) who has a role in the creation of the documentation.
person_id_type A string used to identify where the type ID is located to identify a person.
honorific A string used to identify an honorific for a person_id.
person_name_1 A string used to identify a personal name for a person_id.
person_name_2 A string used to identify a middle name for a person_id.
person_name_3 A string used to identify a family name for a person_id.
person_role An OBI ID used to identify the role a person is involved in, such as an author or corresponding author.
citation A string representing the citation used for a document. This is used as a shorthand to identify locations.
citation_style A string representing the citation style used for a citation.
document_source A string representing a URL or file source for a document.
