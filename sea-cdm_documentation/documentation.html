Documentation

An unsorted table used to identify all documentation (datatypes, protocols, or results).

Documentation Properties

Property Description
idDocumentation An internal ID used to identify Documentation.
StudyID An internal ID used to identify a study that Documentation belongs to.
DocumentName A string used to identify the name of a document.
ForeignID A string used to identify a foreign ID used for a document. This is paired to ForeignSource.
ForeignSource A string used to identify the source of documentation that a Foreign ID belongs to.
PersonID A string used to find a unique identifier of a person (if available) who has a role in the creation of the documentation.
PersonIDType A string used to identify where the type ID is located to identify a person.
Honorific A string used to identify an honorific for a person ID.
PersonName1 A string used to identify a personal name for a person ID.
PersonName2 A string used to identify a middle name for a person ID.
PersonName3 A string used to identify a family name for a person ID.
PersonRole An OBI ID used to identify the role a person is involved in, such as an author or corresponding author.
Citation A string representing the citation used for a document. This is used as a shorthand to identify locations.
CitationStyle A string representing the citation style used for a citation.
DocumentSource A string representing a URL or file source for a document.
