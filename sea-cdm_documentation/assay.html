Assay

An assay is a process that is defined as taking one or more samples and generating data.

Assay Properties

Property Description
assay_id An internal ID used to describe an assay type.
assay_name A string representing the name of an assay.
documentation_id An ID of the documentation table used to identify the reference document describing the process of an assay.
assay_type An assay category used to assign the type of the assay it belongs to. This is used to describe measurement or observation or survey.
sea_assay_id A string representing the OBI ID for which SEA Assay Type it belongs to.
assay_type_id A string representing the OBI ID for an assay.
organism A binary value indicating if an assay necessarily involves an organism.
reagents A list of material IDs used as inputs by an assay that are consumed in the process (Experimental assays only).
platform A list of material IDs used as inputs by an assay that are not consumed (Experimental assays only).
