Occurrence

An occurrence is defined as a history that affects an organism as part of its lifespan. Occurrences include disease processes, pregnancy, and adverse events (lethal or otherwise).

Occurrence Properties

Property Description
occurrence_id An internal identifier used to identify a specific occurrence.
organism_id An identifier that maps to an organism.
occurrence_name A string that lists the reported summary of an occurrence.
occurrence_ID A string that lists the ontology ID used to identify an occurrence.
occurrence_severity An integer that corresponds to the severity of an occurrence using the Common Terminology for Adverse Events (CTCAE). Benign occurrences are rated as 0.
occurrence_onset A double that lists the onset of an occurrence for an organism.
occurrence_onset_unit A string used to identify the unit for InterventionTime.
occurrence_onset_unit_id A string used to identify the ontology ID for TimeUnit.
