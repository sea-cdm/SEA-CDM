Occurrence

An occurrence is defined as a history that affects an organism as part of its lifespan. Occurrences include disease processes, pregnancy, and adverse events (lethal or otherwise).

Occurrence Properties

Property Description Usage
occurrence_id An internal identifier used to identify a specific occurrence. "OCCUR_1"
organism_id An identifier that maps to an organism. "ORGO_1"
occurrence_name A string that lists the reported summary of an occurrence. "Diabetes", "Fever", "Influenza"
occurrence_id A string that lists the ontology ID used to identify an occurrence. DOID_0081062, OAE_0000361, DOID_8469
occurrence_severity An integer that corresponds to the severity of an occurrence using the Common Terminology for Adverse Events (CTCAE). Benign occurrences are rated as 0. 0, 1, 2, 3, 4, 5
occurence_t0 A string used to identify the time 0 reference for onset study. “Organism age”, “Time of enrollment”, “Time of administration”
occurence_to_id A string used to identify the event for Intervention_t0
occurrence_onset A double that lists the onset of an occurrence for an organism. "-7", "7"
occurrence_onset_unit A string used to identify the unit for InterventionTime. "Year"
occurrence_onset_unit_id A string used to identify the ontology ID for occurence_onset_unit_id. "UO_000035"
occurrence_end A double that lists the end of an occurrence for an organism. "-1", "7"
occurrence_end_id A string used to identify the ontology ID for occurence_end_unit_id. "UO_000035"
reference_id A string that corresponds to the occurrence's ID in another dataset. Must be matched with ForeignSource. "ADVR00001"
reference_source_id A string identifier that corresponds to the data source of the corresponding ForeignID. "ImmPort"
