Analysis

An analysis is defined as a process which transforms a data result into another data result. This includes the use of computational algorithms and statistical analysis.

Analysis Properties

Property Description
analysis_id A primary key used to identify a specific analysis.
group_id A foreign key used to identify a specific group used for an analysis.
documentation_id A foreign key providing the documentation_id from the Documentation table that contains the analysis protocol and related information.
input_data A string listing the data types that are used as inputs for a statistical analysis.
input_data_id A string listing the ontology ID for each input_data class.
analysis_name A string representing the name of a type of analysis.
analysis_name_id A string providing an ontology identifier for the analysis name.
reference_id A string used to identify an external reference for the group.
reference_source A string used to identify the source of the reference_id.
