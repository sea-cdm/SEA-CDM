Analysis

An analysis defined as a planned process which transforms a data result into another data result. A data result for SEA CDM is derived from an assay. This includes the use of computational algorithms and statistical analysis.

Analysis Properties

Property Description Usage
analysis_id A primary key used to identify a specific analysis. "ANLYS_1"
group_id A foreign key used to identify a specific group used for an analysis. "GROUP_1"
documentation_id A foreign key used to provide the documentation_id in the Documentation table that provides the analysis protocol and other related information used to describe an analysis. "DOCU_1"
input_data A string listed the data types that are used as inputs for a statistical analysis "VIGET"
input_data_id A string listing the ontology ID for each input_data class. "NGBO_6000408"
file_access A string listing the download link needed to access the file for the input data. “https://zenodo.org/records/7407195”
analysis_name A string representing the name of a type of analysis. "LIMMA"
analysis_name_id A string used to provide an ontology identification of an analysis name id "EFO_0001456", "NGBO_6000408"
reference_id A string used to identify an external reference for the group.
reference_source A string used to identify the source of the reference_id.
