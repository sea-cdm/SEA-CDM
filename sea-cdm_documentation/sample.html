Sample

A sample is the result of an experiment that is collected from an organism and used to generate data. A biosample refers to the tissue or cell culture collected from an organism, while an experimental sample (expsample) is the sample that has been further processed.

Sample Properties

Property Description
sample_id An internal ID used to identify a specific sample.
organism_id For samples that are composed of multiple organisms, a value of ‘0’ is assigned to organism_id and should be queried via group_id.
group_id The group of organism instances that data is collected from. For samples from multiple organisms. Single-organism samples use value '0'.
collection_time A double representing how long from some T0 that an organism was generated.
t0_definition A string representing the reference event that collection time is measured to.
collection_time_unit A string representing the unit used for the collection time.
collection_time_unit_id A string representing the ontology ID for the collection_time_unit.
biosample_reference_source A string representing the documentation source for the biosample ontology ID.
biopsample_reference_id A string representing the ID listed by the biosample in the reference source.
expsample_id A string representing the type of process a sample was used for (e.g., 'swabbing').
expsample_type_id A string representing the ontological ID listed by the expsample type.
expsample_reference_source A string representing the documentation source for the expsample ontology ID.
expsample_reference_id A string representing the ID listed by the expsample in the reference source.
biosample_collection A string representing the collection method of the biosample.
biosample_collection_id A string representing the ontology ID for the biosample collection method.
replicates An integer representing the number of replicates generated for this sample.
comments A string used as a placeholder for additional comments or notes.
