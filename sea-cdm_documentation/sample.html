Sample

A sample is the result of an experiment that is collected from an organism and used to generate data. A biosample refers to the tissue or cell culture collected from an organism, while an experimental sample (exp sample) is the sample that has been further processed.

Sample Properties

Property Description Usage
sample_id An primary id used to identify a specific sample. "SAMP_1"
organism_id A foreign id used to identify a specific organism. For samples that are composed of multiple organisms, a value of ‘0’ is assigned to organism_id and should be queried via group_id. "ORGO_1"
group_id The group of organism instances that data is collected from. This is done for samples that are generated from multiple organisms "GROUP_1, NULL"
collection_time A double representing how long from some T0 that an organism was generated. "3., 14"
t0_definition A string representing the reference event that collection time is measured to. "TIme of Administration", "Time of Enrollment"
collection_time_unit A string representing the unit used for the collection time. "Day", "Month"
collection_time_unit_id A string representing the ontology ID for the collection_time_unit.
biosample_reference_source A string representing the documentation source for the biosample ontology ID. "ImmPort"
biopsample_reference_id A string representing the ID listed by the biosample in the reference source.
expsample_type A string representing the type of process a sample was used for (e.g., 'swabbing'). "sc-RNA seq B Cell"
expsample_type_id A string representing the ontological ID listed by the expsample type.
expsample_reference_source A string representing the documentation source for the expsample ontology ID. "GEO"
expsample_reference_id A string representing the ID listed by the expsample in the reference source.
biosample_collection A string representing the collection method of the biosample. "Blood Draw", "Swabbing"
biosample_collection_id A string representing the ontology id for biosample_collection. method of the sample.
biosample_type A string represent the initial tissue or part of organism a biosample is derived from. "Lung", "Blood"
biosample_type_id A string representing the ontology id for biosample_type_id. method of the sample.
replicates An integer representing the number of replicates generated for this sample. 1
comments A string used as a placeholder for additional comments or notes.
