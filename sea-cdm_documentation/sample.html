Sample

Sample Properties

Property Description
idSample An internal ID used to identify a specific sample.
GroupID The group of organism instances that data is collected from. This is done for samples that are generated from multiple organisms. For samples that have a group size of 1, a value of '0' is assigned for its GroupID.
SampleType A string representing the type of process a sample was used for (e.g., 'swabbing').
SampleTypeID A string representing the ontological ID listed by SampleType.
SampleCollection A string representing the collection method of the sample.
OrganismID For samples that have a specific organism, this field maps to the ID of the organism.
SampleSource A string representing the organism or tissue source that a sample was collected from. For specimen collection, tissue is preferred.
SampleSourceID A string representing the ontological ID listed by the sample source.
SampleDestruction A string representing whether the sample was preserved or destroyed.
OriginalSample An integer representing the original sampleID from which the sample is derived. A sample ID of 1 indicates that this sample was collected from an organism without processing.
Comments A string used as a placeholder for additional comments or notes.
