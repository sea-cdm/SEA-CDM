Groups

A group is defined as a collection of multiple individual instances that share some common experimental features. These instances can belong to multiple organisms (in case of observations) or samples (in case of assays).

Group Properties

Property Description
group_id An primary key used to identify a specific group.
group_type A string representing if a group is about an organism or sample.
composition A value that describes the nature of the group as a collection of instances for the same organism, multiple experimental replicates, or a heterogeneous group.
group_size A value that stores how many individualorganisms are contained within a group. This is done to cl.
reference_id A string used to identify an external reference for the group.
reference_source A string used to identify the source of the ForeignID.
