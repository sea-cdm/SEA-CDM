Groups

A group is defined as a collection of multiple individual instances that share some common experimental features. These instances can belong to multiple organisms (in case of observations) or samples (in case of assays).

Group Properties

Property Description Usage
group_id An primary key used to identify a specific group. "GROUP_1"
group_type A string representing if a group is about an organism or sample. "Organism", "Sample"
group_type_id A string used to identify the ontology id that corresponds to a group type.
composition A value that describes the nature of the group as a collection of instances for the same organism, multiple experimental replicates, or a heterogeneous group. "Organism", "Sample"
group_size A value that stores how many individualorganisms are contained within a group. This is done to cl. "36"
min_group_age A double that records the youngest age of an organism or a sample derived from an organism in a group. 17
min_age_unit A string represent the unit associated with the min_group_age "Day", "Month"
min_age_unit_id A string represent the ontology id min_age_unit.
max_group_age A double that records the youngest age of an organism or a sample derived from an organism in a group. 87
max_age_unit A string represent the unit associated with the max_group_age "Day", "Month"
max_age_unit_id A string represent the ontology id max_age_unit.
reference_id A string used to identify an external reference for the group. "ARM512"
reference_source A string used to identify the source of the refernce_id. "ImmPort"
