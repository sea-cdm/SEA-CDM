Groups

A group is defined as a collection of multiple individual instances that share some common experimental features.

Group Properties

Property Description
idGroup An internal identifier used to identify a specific group.
ExperimentID An integer used to identify an experiment a specific group belongs to.
GroupConsistency A value that describes the nature of the group as a collection of instances for the same organism, multiple experimental replicates, or a heterogeneous group.
GroupSize A value that stores how many individuals are contained within a group.
ForeignID A string used to identify an external reference for the group.
ForeignSource A string used to identify the source of the ForeignID.
