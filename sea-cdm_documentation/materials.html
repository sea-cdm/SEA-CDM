Materials

A material is defined as one specific material that is used as part of an experiment or assay process. Material ID needs to be accessed as part of Intervention, Sample, or Assay Tables. Complex materials, such as cocktails, are identified by an ontologyID with their components arising from other data sources.

Materials Properties

Property Description
idMaterialAn internal identifier used internally to identify a specific material.
OntologyIDA string used to identify the material used as an ontology; may be a string of a query used to generate the subclasses instead.
MaterialNameA string used to name the material.
OrganizationA string used to identify the organization responsible for the materials.
ForeignIDA string used to identify an ID that is used for a material within a different database or source.
ForeignSourceA string used to identify the source for a ForeignID.
