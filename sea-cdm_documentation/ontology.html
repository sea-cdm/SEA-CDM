Ontology

This table contains a list of all ontologies used for a SEA CDM. Ontologies are special form of documentation that is meant to be directly queried by SEA CDM to link IDs to knowledge graphs. Further information is meant to be queried or linked according to work done in SEA CDM.

Ontology Properties

Property Description Usage
ontology_id A primary key used internally to identify a specific ontology. "ONTO_1"
documentation_id An foreign key used to identify the documenation associated with the ontology. "DOCU_1"
ontology_name A string used to provide the common name that is used as an ontology. "Vaccine Ontology"
ontology_iri A string used to identify the ontology IRI that is used for terms within a specific ontology. "http://purl.obolibrary.org/obo/VO"
