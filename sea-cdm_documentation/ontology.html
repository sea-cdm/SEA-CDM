Ontology

This table contains a list of all ontologies used for a SEA CDM. Further information is meant to be queried or linked according to work done in SEA CDM.

Ontology Properties

Property Description
idOntology An internal identifier used internally to identify a specific ontology.
DocumentationID An ID used to the documentation ID that contains the ontology to be queried as part of OSEAN.
OntologyName A string used to provide the common name that is used as an ontology.
OntologyString A string used to identify the ontology IRI that is used for terms within a specific ontology.
