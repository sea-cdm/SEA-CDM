Page with Sidebar Navigation

This page demonstrates a layout with a scrollable sidebar navigation on the left side. The sidebar lists all available pages in Section 1 for easy navigation.

The sidebar will remain visible as you scroll through the content, making it easy for users to jump to different pages at any time.

Key Features

  • Sticky sidebar that stays in view while scrolling
  • Responsive design that adapts to mobile devices
  • Visual indication of the current active page
  • Scrollable list when there are many navigation items

How to Use This Template

To create additional pages with this sidebar layout:

  1. Copy this HTML file to create a new page
  2. Update the title, content, and sidebar active state
  3. Adjust the sidebar links to reflect your site structure
